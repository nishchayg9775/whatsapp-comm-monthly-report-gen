# Whatsapp Comm Monthly Report Gen

React + Tailwind based stock banner generator for monthly report creatives.

## Features

- Fixed 800x400 export canvas
- Manual and CSV-driven banner generation
- Auto column mapping with fallback manual mapping
- Category-wise top performer extraction
- Multiple built-in visual templates
- Editable text, card colors, layout, and logo placement
- PNG export

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## GitHub Pages

This project is configured to deploy automatically to GitHub Pages using the workflow in:

`/.github/workflows/deploy-pages.yml`
